# 🚀 DevMatch-AI 
### A Developer Networking Platform (Work in Progress)

DevMatch-AI is a **full-stack web application** aimed at connecting developers through profiles, content exploration, and (upcoming) social interactions like posts, likes, and follows.

>  **Status:** Actively under development. Features are being added incrementally.

---

##  Features

###  Implemented
-  User authentication (Register / Login)
-  Developer profiles
-  Explore page with AI-curated articles & GitHub repositories
-  Backend AI utility setup (experimental)
-  Sidebar-based navigation layout
-  MongoDB schema design for users & posts

###  In Progress / Planned
-  User-generated posts from frontend
-  Like system (ObjectId-based)
-  Follow / Following relationships
-  Feed showing posts from followed users
-  Smarter AI recommendations
-  UI/UX improvements

---

## 🛠 Tech Stack

**Frontend**
- EJS
- HTML, CSS, JavaScript

**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)

**Tools & Services**
- OpenAI (AI utilities – experimental)
- Git & GitHub
- Render (Deployment)

##  Project Structure (Simplified)

   git clone https://github.com/your-username/devmatch.ai.git
   cd devmatch.ai
2️⃣ Install dependencies
   npm install

   MONGO_URI=your_mongodb_connection_string
